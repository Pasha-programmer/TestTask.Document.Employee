# TestTask.Document.Employee

1) Открыть Visual Studio
2) Запустить Docker
3) Выбрать параметры запуска под Docker Compose
4) Запустить приложение
5) Открыть http://localhost:5052/swagger
Будет предоставлен open api swagger, для тестирования системы.
Есть идентификация пользователя, для этого нужно получить токен авторизации, где логин - это имя пользователя, а пароль - его роль, если роль = "Accountant", то роль бухгалтера, иначе роль обычного сотрудника. Далее можно пользователя другими эндпоинтами.
